## React Hooks
 | Name       | Goal                                                  |
 | ---------- | ----------------------------------------------------- |
 | useState   | allow function component ot use component-level state |
 | useEffect  | use life-cycle methods                                |
 | useContext | use context system                                    |
 | useRef     | use ref system                                        |
 | useReducer | store data through a 'reducer'                        |
